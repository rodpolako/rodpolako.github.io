Chess Tactics Extractor

Enter in the Lichess User ID below and this tool will download all the matching Lichess games that have been computer analyzed in Lichess.

Once downloaded, this tool will generate a PGN with puzzles based on moves the Lichess engine has indicated as a problem along with its suggested moves. At this time, it only will work with standard chess games (no variants).
